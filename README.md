# How to Install

## Chrome, Brave or Edge
1. Download the source code
2. Go into the Chrome, Brave or Edge extension manager
3. Enable Developer Mode by clicking the switch next to Developer Mode in the top right
4. Click load unpacked
5. Go into the source, go and select the folder named "sagewood-theme", them click open
6. Enjoy your new theme
